BProgram Essentials

사업 핵심

B.01
대상 기업 / Eligibility
서울 소재
수출 중소기업
본점이 서울시에 소재하고 전년도 수출실적 5천만 달러 이하인 중소기업. 수출 보험·보증을 이용(예정)하는 기업.
B.02
지원 규모 / Scale
보험·보증료
연 300만원
업체당 연간 300만원 한도 지원(중동·북아프리카 21개국 수출 시 800만원). 보험 가입이 아닌 보험·보증료를 대납·지원하는 사업.
B.03
신청 방식 / Method
K-SURE 통해
신청·심사
신청서 작성 → 한국무역보험공사 제출 → 심사 → 보험·보증료 지원. 신청서 및 붙임서류는 최초 1회만 제출하면 됨.
세부 지원 종목
Covered Products
종목군종수내용
수출보험6종수입자로부터 수출대금 미회수 시 손실 보상 (단기수출보험·다이렉트·중소Plus+·서비스종합·단체보험 등)
수출신용보증6종수출금융 이용에 따라 은행에 부담하는 상환채무 연대 보증 (선적전·선적후·NEGO·매입·포괄매입 등)
환변동보험1종계약시점과 대금회수 시점의 환율 차이로 발생하는 환위험 제거
문화산업보증1종문화상품 수출기업이 수출자금 대출 시 은행 상환채무 연대 보증
반드시 짚을 핵심Key Notes
  • 보험 가입이 아닌 보험·보증료 자체를 지원하는 사업 (수출자가 보험·보증에 가입)
  • 예산(16억) 소진 또는 업체별 한도 초과 시 별도 통보 없이 지원 중단 — 수출자 부담 전환
  • 중동·북아프리카 21개국 수출 중소기업은 한도 800만원으로 상향 (사태 종료 시까지)
  • 지원신청서·붙임서류는 최초 1회만 제출 (반복 신청 불요)
  • 사업개시 전 발생분 소급지원 신청기한(2026.4.30)은 이미 종료

  • 본점이 서울특별시에 소재한 중소기업

지원 방식 — 유의 사항 / Notes
  • 보험·보증료 지원 사업 (보험 가입·부보는 수출자가 별도 진행)
  • 예산 소진·한도 초과 시 별도 통보 없이 지원 중단
  • 기재내용 허위 판명 시 지원금 환급
우대 (한도 상향) / Enhanced
  • 중동·북아프리카 21개국 수출 중소기업 — 연간 한도 800만원(아랍에미리트·사우디·카타르·이집트·모로코 등)

신청경로 (K-SURE ON)How to Apply

  • ① 일반 상품: K-SURE ON 로그인 → 보험/보증료 → 보험료 지원사업 → 신규신청 → 지원기관 ‘서울특별시청’ 선택 → 서류 업로드
  • ② 단체보험: 보험가입 → 단체보험 → 신규신청 → 단기수출보험(단체보험) 일반, 계약자 ‘서울특별시청’ 선택 → 서류 업로드 (또는 이메일 gksure@ksure.or.kr)
  • 접수: 본사(중앙지사) 02-399-6851 / 강남지사 02-551-0473 / 구로디지털지사 02-6911-6910
